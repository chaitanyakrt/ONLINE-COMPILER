# ONLINE-COMPILER
<h2>This is a Full stack application, that uses the Online platform to compile a given code and gives the output(c/c++) built using MERN stack</h2>
<h2>The main features of the app are having an user authentication, user friendly interaction and a proper database(MongoDB Atlas) to store the user information</h2>
<h2>The frontend part of the application is in still development, the css files are yet need to be updated</h2>
