
# ONLINE-COMPILER

## This is a Full stack application, that uses the Online platform to compile a given code and gives the output(c/c++) built using MERN stack
## The main features of the app are having an user authentication, user friendly interaction and a proper database(MongoDB Atlas) to store the user information
## The frontend part of the application is in still development, the css files are yet need to be updated
